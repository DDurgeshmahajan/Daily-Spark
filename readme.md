# Daily Spark

## Short Description
Daily Spark is a simple console-based application designed to help you store and display inspiring quotes and affirmations. Perfect for daily boosts of motivation! 

## Features
- **Quote Storage:** Stores quotes in a plain text file.
- **Random Display:** Shows a random quote each time you request.
- **Add New Quote:** Allows you to add new quotes to the collection.
- **List All Quotes:** Displays all stored quotes one by one.
- **Interactive Menu:** Easy-to-use text-based menu for navigation.
- **Error Handling:** Manages errors like missing files or invalid inputs gracefully.

## Usage Instructions
1. **Download the Project:**
   - Clone or download the repository to your local machine.

2. **Run the Application:**
   - Open your terminal or command prompt.
   - Navigate to the project directory.
   - Run the application using Python: `python daily_spark.py`

3. **Using the Menu:**
   - **1. Get a Quote:** Displays a random quote.
   - **2. Add a Quote:** Prompts to add a new quote.
   - **3. List All Quotes:** Shows all stored quotes.
   - **4. Exit:** Closes the application.

## Dependencies
- Python 3.x (Any version of Python 3 will work)
